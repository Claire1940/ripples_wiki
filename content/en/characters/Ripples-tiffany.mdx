<Section icon={Shield} title="Technical Guide: Fixing Android Image Issues">

A common hurdle for players following the **Ripples Tiffany** route on mobile devices is the "Gray Screen" or "Missing Images" bug. This typically occurs when the game's asset files, specifically the `archive.rpa`, are not correctly mapped by the Android OS. Following the Episode 9 update, a specific file structure is required to ensure Tiffany's high-definition renders load correctly.

<Steps>
  <Step num="1" title="Download the Correct Assets">
    Ensure you have the latest .apk and the separate .rpa files provided by the developer on itch.io or Patreon. The mobile version often requires manual file placement.
  </Step>
  <Step num="2" title="Locate the Game Folder">
    Use a file manager (like ZArchiver) to find the `Android/data/com.jestur.ripples/files/game` directory. This is where the core assets must reside.
  </Step>
  <Step num="3" title="Replace the Archive File">
    Delete any existing `archive.rpa` in the game folder and copy the new version into its place. Ensure the file name is exactly `archive.rpa` with no extra extensions.
  </Step>
  <Step num="4" title="Verify File Permissions">
    Go to your phone's App Settings, find Ripples, and ensure "Files and Media" access is granted. Without this, the game cannot read the textures for Tiffany's scenes.
  </Step>
</Steps>

<Callout type="success" title="Resolved Issues">
Users who follow these steps correctly report that the gray screens are replaced with the intended 3D animations. If you still encounter issues, try clearing the app cache before restarting the game.
</Callout>

</Section>
