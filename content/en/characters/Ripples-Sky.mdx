<Section icon={Shield} title="Step-by-Step Android Installation & Save Transfer Guide">

Playing the game on Android offers a convenient way to experience the story on the go. However, because the game is distributed via itch.io as an APK file, users must follow specific steps to install the game and transfer their existing save data from PC to mobile.

Follow these steps to ensure a smooth installation and save migration process.

<Steps>
  <Step num="1" title="Enable Unknown Sources">
    Navigate to your Android device settings, select Security or Privacy, and enable the option to install apps from unknown sources. This allows the installation of APK files downloaded outside of the official Google Play Store.
  </Step>
  <Step num="2" title="Download the Official APK">
    Visit the official itch.io page using your mobile browser. Download the latest Android APK file provided by Jestur. Ensure you have at least 5 GB of free storage space before starting the download.
  </Step>
  <Step num="3" title="Install the Game">
    Locate the downloaded APK file in your device's file manager. Tap on the file and follow the on-screen prompts to complete the installation. Once installed, launch the game once to generate the necessary system folders.
  </Step>
  <Step num="4" title="Transfer Save Files (Optional)">
    To transfer saves from PC to Android, connect your phone to your computer. Copy the save files from the PC directory (`%APPDATA%/RenPy/Ripples/`) to the corresponding Android folder (`Android/data/com.jestur.ripples/files/saves/`).
  </Step>
</Steps>

Following these steps ensures that you do not lose your progress when switching between platforms or upgrading to the latest Episode 9 build.

<Callout type="success" title="Save Compatibility">
Save files generated on PC are fully compatible with the Android version, provided both platforms are running the exact same version of the game (e.g., Episode 9).
</Callout>
