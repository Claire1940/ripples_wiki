<Section icon={Target} title="Step-by-Step Installation Guide">

Proper installation of Ripples 0.9.22 ensures that game assets load correctly and existing story progress is preserved. Because the game is distributed as a standalone package rather than through automated platforms, players must manage files manually. This is particularly important for Android users and desktop players migrating saves from previous episodes.

The game is built on the Ren'Py engine, which stores user configuration and save files in platform-specific directory paths. Understanding these locations prevents accidental data loss during version upgrades.

<Steps>
  <Step num="1" title="Download the Package">
    Visit the official itch.io page or Jestur's Patreon. Select the appropriate archive file matching your operating system. Ensure the download completes fully before attempting to open the package.
  </Step>
  <Step num="2" title="Prepare the Directory">
    For desktop systems, create a clean directory away from system folders to avoid permission issues. For Android devices, ensure at least 6 GB of free internal storage space is available before starting.
  </Step>
  <Step num="3" title="Extract or Install">
    On Windows, extract the .ZIP archive using standard extraction software. On Android, locate the downloaded .APK file using a file manager and tap to initiate the package installer.
  </Step>
  <Step num="4" title="Configure Security Permissions">
    On macOS, you may need to bypass Gatekeeper by right-clicking the application and selecting Open. On Android, allow installation from Unknown Sources in your security settings if prompted.
  </Step>
  <Step num="5" title="Verify Save State Migration">
    Launch the game. The engine should automatically detect existing save files from previous directory locations. Verify your progress by loading a recent save point from the main menu.
  </Step>
</Steps>
