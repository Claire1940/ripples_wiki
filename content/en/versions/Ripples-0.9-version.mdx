<Section icon={Shield} title="Technical Troubleshooting & Save Migration">

One of the most common issues players face when updating to the **Ripples 0.9 version** is the loss of previous save data. Because visual novels rely heavily on continuity, maintaining your progress across updates is paramount. The game utilizes the Ren'Py engine, which stores save files in a standardized location across different operating systems.

If you encounter technical errors or missing saves, consult the table below for standard troubleshooting procedures:

| Technical Issue | Potential Cause | Recommended Solution |
| :--- | :--- | :--- |
| Missing Saves | Folder path mismatch | Copy files from the old Ren'Py directory to the new directory |
| Game Crashes on Launch | Corrupted installation files | Delete the game folder and perform a clean extraction |
| Android Installation Error | Package conflict | Uninstall the older APK version before installing the 0.9 APK |
| Performance Lag | High-res image caching | Lower the image cache limit in the in-game settings menu |

<Callout type="warning" title="Save Backup Warning">
Always create a backup copy of your save directory before installing any new update. A simple system crash during the update process can permanently corrupt your progress files.
</Callout>

To find your save files manually, navigate to the following system paths depending on your operating system:
- **Windows**: `%APPDATA%/RenPy/Ripples-`
- **macOS**: `~/Library/RenPy/Ripples-`
- **Android**: `Android/data/com.jestur.ripples/files/saves/`

</Section>
