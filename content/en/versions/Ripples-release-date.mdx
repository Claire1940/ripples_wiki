<Section icon={Target} title="How to Download and Update">

Transitioning between episodes requires proper installation practices to prevent the loss of player choices. Since *Ripples* relies on decisions made in early chapters to determine late-game outcomes, preserving your save files is critical.

<Steps>
  <Step num="1" title="Visit the Official Distribution Pages">
    Navigate to the official itch.io page for Ripples or access the creator's Patreon page. Patreon supporters often receive early access to upcoming builds before they are released to the general public on itch.io.
  </Step>
  <Step num="2" title="Select the Correct Build">
    Download the specific package designed for your operating system. Ensure you have sufficient storage space, as file sizes increase with each new episode.
  </Step>
  <Step num="3" title="Extract and Install the Game Files">
    For PC platforms, extract the downloaded compressed folder to a dedicated directory. Do not run the game directly from within a zipped folder, as this prevents save files from writing correctly. For Android, enable installation from unknown sources in your device settings before opening the APK.
  </Step>
  <Step num="4" title="Verify Save File Migration">
    The game engine automatically detects existing save folders in the default system directory. Launch the new episode and check the load menu to confirm that your progress from previous episodes has successfully carried over.
  </Step>
</Steps>

<Callout type="tip" title="Manual Backup Recommendation">
Before installing a new episode update, manually copy your save directory to a secure backup folder. On Windows, save data is typically located in the AppData/Roaming/RenPy directory under the game's unique folder name.
</Callout>

</Section>
