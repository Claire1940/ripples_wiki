<Section icon={Shield} title="Android Installation & Technical Setup Guide">

The mobile version of the **Ripples game** allows players to enjoy the visual novel on Android devices. Because the game is distributed as an APK file via itch.io, players must manually install and configure the application on their devices.

Follow these structured steps to install the game safely and manage your save files without losing progress.

<Steps>
  <Step num="1" title="Download the Official APK">
    Visit the official itch.io page using your Android device's web browser. Download the official Episode 9 Android APK file. Avoid downloading from unauthorized forums or third-party file-sharing sites.
  </Step>
  <Step num="2" title="Enable Unknown Sources">
    Open your Android device's settings menu. Navigate to Security or Applications, and enable the option to install apps from "Unknown Sources." This permission is required to install applications outside of the standard app store.
  </Step>
  <Step num="3" title="Install the Application">
    Locate the downloaded APK file using a file manager app. Tap the file and select "Install." Wait for the installation process to complete, which may take a few moments due to the large volume of high-resolution images.
  </Step>
  <Step num="4" title="Transfer Save Files (Optional)">
    If you are upgrading from an older episode, locate your previous save files. Copy them from the old game folder and paste them into the newly created Episode 9 save directory to continue your progress.
  </Step>
</Steps>
