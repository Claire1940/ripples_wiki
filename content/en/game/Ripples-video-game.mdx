### Android Installation Steps

To play on mobile devices, follow these steps to install the official APK package securely:

1. Navigate to the official itch.io page for the game.
2. Download the latest Android APK file (optimized for Episode 9).
3. Open your Android device settings, go to Security, and enable **Install Unknown Apps** for your browser or file manager.
4. Locate the downloaded APK file in your device's downloads folder and tap to install.
5. Launch the game and grant necessary storage permissions to allow save files to write correctly.
