<Section icon={Shield} title="Technical Guide: Installing Ripples Ep 9 on Android & PC">

*Ripples* is built on the Ren'Py engine, making it highly compatible across multiple platforms, including Windows, macOS, Linux, and Android. However, updating to a new episode can sometimes present technical challenges, particularly regarding save file migration and directory structures.

The table below outlines the system requirements and file structures for both major platforms to ensure a smooth installation process.

| Platform | Minimum OS | Storage Required | Default Save Directory |
| :--- | :--- | :--- | :--- |
| **PC (Windows)** | Windows 7 or higher | Approx. 2.5 GB | `%APPDATA%/RenPy/Ripples-Jestur` |
| **Android** | Android 8.0 or higher | Approx. 2.2 GB | `/Android/data/com.jestur.ripples/files` |

<Callout type="tip" title="Android Save Migration">
On Android, operating system security updates can sometimes restrict access to the `Android/data` folder. To manually backup or transfer your saves from Episode 8 to Episode 9, use a third-party file manager that supports scoped storage access, or utilize the in-game cloud save feature if supported by your distribution platform.
</Callout>

If you encounter performance issues, such as lagging transitions or slow image loading, navigate to the game's preferences menu and adjust the following settings:

- **Image Cache Size**: Set to high if your device has at least 4 GB of RAM.
- **Transitions**: Disable complex screen transitions to improve performance on older Android devices.
- **Skip Unseen Text**: Ensure this is toggled off to avoid accidentally skipping new Episode 9 narrative content.

Always download updates directly from official sources like Jestur's itch.io page or authorized Patreon links to avoid corrupted files or security risks.
