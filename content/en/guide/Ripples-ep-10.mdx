<Section icon={Shield} title="Save File Management and Platform Setup">

Because this visual novel is built on the Ren'Py engine, your save files are stored in directory folders that are separate from the main game files. This design ensures that when you download and install a new version, your progress is automatically detected. However, platform migrations or clean installations can sometimes risk data loss if not handled carefully.

To guarantee a seamless transition when upgrading to the next release, players should familiarize themselves with the manual backup process. This is particularly important for those playing across multiple devices or utilizing Android devices.

<Steps>
  <Step num="1" title="Locate Your Save Directory">
    On Windows, navigate to the AppData folder by typing `%appdata%/RenPy` into the file explorer. Locate the folder associated with the game to find your persistent data.
  </Step>
  <Step num="2" title="Create a Backup Copy">
    Copy the entire save folder to a secure location on your desktop or cloud storage. This acts as a safety net in case of file corruption during the update process.
  </Step>
  <Step num="3" title="Download the Update">
    Once the next update is officially released, download the appropriate package for your operating system (Windows, macOS, Linux, or Android) from the official itch.io page.
  </Step>
  <Step num="4" title="Install and Verify">
    Extract the new game files and launch the executable. The game will automatically read your existing save files, allowing you to resume your story immediately.
  </Step>
</Steps>

For mobile players, managing save files requires extra precaution. Android operating systems often restrict access to internal app directories, making manual backups slightly more complex.

| Platform | Save Location Path | Recommended Backup Tool |
| | | |
| Windows | `%appdata%/RenPy/Ripples` | Windows File Explorer |
| macOS | `~/Library/RenPy/Ripples` | Finder |
| Linux | `~/.renpy/Ripples` | Terminal / File Manager |
| Android | `Internal Storage/Android/data` | Specialized File Manager App |

<Callout type="warning" title="Android Update Warning">
When updating the game on Android, do not uninstall the previous version unless explicitly instructed by the developer. Uninstalling the application will delete the corresponding app data folder, which contains your save files. Instead, install the new APK over the existing installation to overwrite the game files while preserving your progress.
</Callout>

</Section>
