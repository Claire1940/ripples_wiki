<Section icon={Shield} title="Руководство по установке на Android и технической настройке">

Мобильная версия **игры Ripples** позволяет игрокам наслаждаться визуальной новеллой на устройствах Android. Поскольку игра распространяется в виде файла APK через itch.io, игрокам необходимо вручную установить и настроить приложение на своих устройствах.

Следуйте этим пошаговым инструкциям, чтобы безопасно установить игру и управлять файлами сохранений без потери прогресса.

<Steps>
  <Step num="1" title="Скачайте официальный APK">
    Посетите официальную страницу на itch.io через веб-браузер вашего Android-устройства. Скачайте официальный файл APK Эпизода 9 для Android. Избегайте скачивания с неавторизованных форумов или сторонних файлообменников.
  </Step>
  <Step num="2" title="Разрешите установку из неизвестных источников">
    Откройте меню настроек вашего Android-устройства. Перейдите в раздел «Безопасность» или «Приложения» и включите параметр установки приложений из «Неизвестных источников». Это разрешение требуется для установки приложений вне стандартного магазина приложений.
  </Step>
  <Step num="3" title="Установите приложение">
    Найдите загруженный файл APK с помощью приложения-файлового менеджера. Нажмите на файл и выберите «Установить». Дождитесь завершения процесса установки, который может занять некоторое время из-за большого объема изображений высокого разрешения.
  </Step>
  <Step num="4" title="Перенесите файлы сохранений (необязательно)">
    Если вы обновляетесь с более старого эпизода, найдите ваши предыдущие файлы сохранений. Скопируйте их из старой папки игры и вставьте в только что созданную директорию сохранений Эпизода 9, чтобы продолжить прохождение.
  </Step>
</Steps>

| Платформа | Минимальная версия ОС | Рекомендуемое место | Формат файла |
| --- | --- | --- | --- |
| Android | Android 8.0 (Oreo) | 3 ГБ свободного места | APK |
| Windows | Windows 7 или выше | 4 ГБ свободного места | ZIP / EXE |
| macOS | macOS 10.12 или выше | 4 ГБ свободного места | APP / DMG |
| Linux | Ubuntu 18.04 или эквивалент | 4 ГБ свободного места | Архив tar.gz / Исполняемый файл |

<Callout type="info" title="Стандартное расположение сохранений">
На Android файлы сохранений обычно хранятся во внутренней памяти по пути `Android/data/com.jestur.ripples/files/saves`. Для пользователей Windows сохранения находятся в папке `game/saves` внутри основной директории игры или в локальной папке `AppData/Roaming`.
</Callout>
