<Section icon={Shield} title="Управление файлами сохранений и настройка платформ">

Поскольку эта визуальная новелла создана на движке Ren'Py, ваши файлы сохранений хранятся в отдельных папках, не связанных с основными файлами игры. Благодаря этому при загрузке и установке новой версии ваш прогресс определяется автоматически. Тем не менее, перенос на другую платформу или чистая установка иногда могут привести к потере данных, если не соблюдать осторожность.

Чтобы гарантировать плавный переход при обновлении до следующей версии, игрокам рекомендуется ознакомиться с процессом резервного копирования вручную. Это особенно важно для тех, кто играет на нескольких устройствах или использует Android-устройства.

<Steps>
  <Step num="1" title="Найдите папку с сохранениями">
    На Windows перейдите в папку AppData, введя `%appdata%/RenPy` в проводнике. Найдите папку, связанную с игрой, чтобы найти ваши постоянные данные.
  </Step>
  <Step num="2" title="Создайте резервную копию">
    Скопируйте всю папку сохранений в безопасное место на рабочем столе или в облачном хранилище. Это послужит подстраховкой на случай повреждения файлов в процессе обновления.
  </Step>
  <Step num="3" title="Скачайте обновление">
    Как только следующее обновление будет официально выпущено, скачайте соответствующий пакет для вашей операционной системы (Windows, macOS, Linux или Android) с официальной страницы itch.io.
  </Step>
  <Step num="4" title="Установите и проверьте">
    Распакуйте новые файлы игры и запустите исполняемый файл. Игра автоматически считает ваши существующие файлы сохранений, позволяя сразу же продолжить историю.
  </Step>
</Steps>

Для мобильных игроков управление файлами сохранений требует особой осторожности. Операционные системы Android часто ограничивают доступ к внутренним каталогам приложений, что делает ручное резервное копирование несколько более сложным.

| Платформа | Путь к папке сохранений | Рекомендуемый инструмент резервного копирования |
| | | |
| Windows | `%appdata%/RenPy/Ripples` | Проводник Windows |
| macOS | `~/Library/RenPy/Ripples` | Finder |
| Linux | `~/.renpy/Ripples` | Терминал / Файловый менеджер |
| Android | `Внутренняя память/Android/data` | Специализированное приложение файлового менеджера |

<Callout type="warning" title="Предупреждение при обновлении на Android">
При обновлении игры на Android не удаляйте предыдущую версию, если это прямо не предписано разработчиком. Удаление приложения сотрет соответствующую папку данных приложения, в которой хранятся ваши файлы сохранений. Вместо этого установите новый APK поверх существующей версии, чтобы перезаписать файлы игры с сохранением вашего прогресса.
</Callout>
