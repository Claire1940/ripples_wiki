<Section icon={Shield} title="Технические советы по управлению сохранениями">

Поскольку *Ripples* доступна на Windows, macOS, Linux и Android, управление сохранениями для разных веток имеет ключевое значение. Движок игры поддерживает множество слотов для сохранений, что крайне важно для проверки того, какая ветка — Дафны или Кристалл — больше соответствует вашим вкусам, без потери текущего прогресса.

| Платформа | Место сохранения | Метод резервного копирования |
|:---|:---|:---|
| **Windows** | %APPDATA%/Roaming/Ripples/saves | Ручное копирование папки |
| **Android** | Android/data/com.jestur.ripples/files | Облачная синхронизация или файловый менеджер |
| **macOS** | ~/Library/Application Support/Ripples | Time Machine или вручную |
| **Linux** | ~/.local/share/Ripples | Стандартный бэкап через tar |
