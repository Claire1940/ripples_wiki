<Section icon={Shield} title="Descarga oficial y pasos de instalación en Android">

Hacer funcionar el juego en tu plataforma preferida es sencillo. El desarrollador, Jestur, ofrece versiones optimizadas tanto para sistemas operativos de escritorio como para dispositivos móviles con Android. Dado que el juego cuenta con más de 10 000 imágenes de alta calidad, es fundamental asegurarse de tener suficiente espacio de almacenamiento antes de comenzar el proceso de instalación.

Sigue estas instrucciones paso a paso para instalar el juego en tu plataforma preferida.

<Steps>
  <Step num="1" title="Visita el centro oficial">
    Navega a la página oficial de **Ripples itch io** o al perfil de Patreon de Jestur. Esto garantiza que estés descargando la versión auténtica y sin modificar del juego.
  </Step>
  <Step num="2" title="Selecciona la versión para tu plataforma">
    Elige el archivo de descarga adecuado para tu dispositivo. Los usuarios de escritorio pueden seleccionar entre archivos zip para Windows, macOS o Linux, mientras que los usuarios de dispositivos móviles deben descargar el archivo APK.
  </Step>
  <Step num="3" title="Extraer y ejecutar (PC/Mac)">
    Para sistemas de escritorio, extrae la carpeta ZIP descargada en un directorio dedicado. Ejecuta el archivo ejecutable (`Ripples.exe` en Windows o el paquete de la aplicación en macOS) para iniciar el juego.
  </Step>
  <Step num="4" title="Instalar el APK (Android)">
    En tu dispositivo Android, habilita "Instalar desde fuentes desconocidas" en los ajustes de seguridad. Localiza el archivo APK descargado usando un administrador de archivos y tócalo para comenzar la instalación.
  </Step>
</Steps>

| Plataforma | Versión mínima de SO | Almacenamiento recomendado | RAM recomendada |
| :--- | :--- | :--- | :--- |
| **Windows** | Windows 7 o superior | **6 GB** de espacio disponible | 4 GB |
| **Android** | Android 5.0 o superior | **6 GB** de espacio disponible | 3 GB |
| **macOS** | High Sierra o superior | **6 GB** de espacio disponible | 4 GB |
| **Linux** | Ubuntu 18.04 o superior | **6 GB** de espacio disponible | 4 GB |

<Callout type="success" title="Migración exitosa de archivos de guardado">
Al actualizar de un episodio anterior al Episodio 9, tus archivos de guardado deberían transferirse automáticamente si instalas la nueva versión en el mismo directorio (en PC) o sobrescribes la aplicación existente (en Android). Siempre haz una copia de seguridad manual de tu carpeta de guardados por si acaso.
</Callout>
