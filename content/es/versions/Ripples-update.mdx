<Section icon={Target} title="Cómo instalar la actualización de Ripples en todas las plataformas">

Instalar la última actualización requiere descargar la versión correcta para tu sistema operativo específico. El desarrollador ofrece soporte oficial para Windows, macOS, Linux y Android. Para proteger tu sistema y apoyar al creador, descarga siempre los archivos del juego desde fuentes oficiales como itch.io o Patreon. Evita descargar archivos de servidores espejo no oficiales de terceros, ya que pueden contener malware o recursos de juego dañados que podrían corromper tus archivos de guardado.

La siguiente tabla proporciona los detalles esenciales para cada plataforma compatible, incluidos los requisitos del sistema operativo y las ubicaciones predeterminadas de los archivos de guardado.

| Plataforma | SO compatible | Método de instalación | Ubicación del archivo de guardado |
| :--- | :--- | :--- | :--- |
| Windows | Windows 10/11 | Extraer ZIP y ejecutar EXE | AppData/Local/Ripples |
| macOS | macOS Big Sur o posterior | Extraer DMG y arrastrar a Aplicaciones | Library/Application Support/Ripples |
| Linux | Basado en Ubuntu/Debian | Extraer tar.gz y ejecutar ejecutable | .config/Ripples |
| Android | Android 8.0 o superior | Instalar APK oficial | Almacenamiento interno/Android/data |

Para los jugadores de dispositivos móviles, instalar la versión de Android requiere unos pasos específicos para garantizar que el juego funcione sin problemas y acceda a tu progreso existente. Sigue esta guía de instalación paso a paso para configurar la actualización en tu dispositivo Android.

<Steps>
  <Step num="1" title="Descargar el APK oficial">
    Visita la página oficial de itch.io o Patreon en tu navegador móvil y descarga el archivo APK de Android más reciente del Episodio 9.
  </Step>
  <Step num="2" title="Habilitar fuentes desconocidas">
    Dirígete a los ajustes de tu dispositivo Android, selecciona Seguridad o Privacidad, y habilita el permiso para instalar aplicaciones de fuentes desconocidas.
  </Step>
  <Step num="3" title="Instalar y sobrescribir">
    Busca el APK descargado usando tu gestor de archivos. Toca el archivo para comenzar la instalación. Si tienes una versión anterior instalada, elige actualizar y sobrescribir la aplicación existente para conservar tus datos de guardado.
  </Step>
  <Step num="4" title="Verificar permisos">
    Inicia el juego y concede los permisos de almacenamiento si se te solicita. Esto asegura que el juego pueda leer y escribir archivos de guardado en el almacenamiento local de tu dispositivo.
  </Step>
</Steps>

<Callout type="warning" title="Evitar la pérdida de archivos de guardado">
Nunca desinstales la versión anterior del juego en Android antes de instalar la actualización. Realizar una desinstalación limpia eliminará permanentemente tu carpeta local de guardado. Realiza siempre una instalación de sobrescritura para mantener tu progreso intacto.
</Callout>

</Section>

<Section icon={Shield} title="Optimización técnica y resolución de problemas">

Ejecutar una novela visual que cuenta con más de 10,000 imágenes de alta calidad puede provocar ocasionalmente problemas de rendimiento, particularmente en dispositivos móviles con almacenamiento o RAM limitados. Asegurarte de que tu dispositivo esté optimizado correctamente evitará fallos y errores visuales durante el juego.

Si encuentras dificultades técnicas después de instalar la última actualización, consulta la siguiente tabla de resolución de problemas para obtener soluciones rápidas a los problemas comunes.

| Descripción del problema | Causa potencial | Solución recomendada |
| :--- | :--- | :--- |
| El juego se cierra al iniciar | RAM insuficiente o archivos corruptos | Borra la caché, reinicia el dispositivo o vuelve a descargar el paquete |
| Los archivos de guardado no cargan | Faltan permisos o cambio de directorio | Concede acceso al almacenamiento manualmente en los ajustes de la aplicación |
| Faltan imágenes de fondo | Extracción incompleta de archivos ZIP | Vuelve a extraer los archivos del juego usando una herramienta oficial como 7-Zip |

Para mantener un rendimiento óptimo en todas las plataformas, considera implementar las siguientes mejores prácticas:

- Cierra las aplicaciones en segundo plano para liberar RAM antes de iniciar el juego.
- Asegúrate de que tu dispositivo tenga al menos 3 GB de espacio de almacenamiento interno libre para acomodar la caché de imágenes de alta resolución.
- Realiza copias de seguridad de tu directorio de guardado con regularidad en una unidad de almacenamiento externa o servicio en la nube.
