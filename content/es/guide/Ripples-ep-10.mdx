<Section icon={Shield} title="Gestión de archivos de guardado y configuración de plataformas">

Debido a que esta novela visual está desarrollada con el motor Ren'Py, tus archivos de guardado se almacenan en carpetas de directorio que están separadas de los archivos principales del juego. Este diseño garantiza que cuando descargues e instales una nueva versión, tu progreso se detecte automáticamente. Sin embargo, las migraciones de plataforma o las instalaciones limpias a veces pueden poner en riesgo la pérdida de datos si no se manejan con cuidado.

Para garantizar una transición sin problemas al actualizar al próximo lanzamiento, los jugadores deben familiarizarse con el proceso de respaldo manual. Esto es particularmente importante para quienes juegan en múltiples dispositivos o utilizan dispositivos Android.

<Steps>
  <Step num="1" title="Localiza tu directorio de guardado">
    En Windows, navega a la carpeta AppData escribiendo `%appdata%/RenPy` en el explorador de archivos. Localiza la carpeta asociada con el juego para encontrar tus datos persistentes.
  </Step>
  <Step num="2" title="Crea una copia de seguridad">
    Copia toda la carpeta de guardado en una ubicación segura en tu escritorio o almacenamiento en la nube. Esto actúa como una red de seguridad en caso de corrupción de archivos durante el proceso de actualización.
  </Step>
  <Step num="3" title="Descarga la actualización">
    Una vez que se lance oficialmente la próxima actualización, descarga el paquete adecuado para tu sistema operativo (Windows, macOS, Linux o Android) desde la página oficial de itch.io.
  </Step>
  <Step num="4" title="Instala y verifica">
    Extrae los nuevos archivos del juego e inicia el ejecutable. El juego leerá automáticamente tus archivos de guardado existentes, lo que te permitirá reanudar tu historia de inmediato.
  </Step>
</Steps>

Para los jugadores de dispositivos móviles, gestionar los archivos de guardado requiere una precaución adicional. Los sistemas operativos Android a menudo restringen el acceso a los directorios internos de las aplicaciones, lo que hace que los respaldos manuales sean un poco más complejos.

| Plataforma | Ruta de ubicación de guardado | Herramienta de respaldo recomendada |
| | | |
| Windows | `%appdata%/RenPy/Ripples` | Explorador de archivos de Windows |
| macOS | `~/Library/RenPy/Ripples` | Finder |
| Linux | `~/.renpy/Ripples` | Terminal / Administrador de archivos |
| Android | `Almacenamiento interno/Android/data` | Aplicación especializada de administración de archivos |

<Callout type="warning" title="Advertencia de actualización en Android">
Al actualizar el juego en Android, no desinstales la versión anterior a menos que el desarrollador lo indique explícitamente. Desinstalar la aplicación eliminará la carpeta de datos de la aplicación correspondiente, que contiene tus archivos de guardado. En su lugar, instala el nuevo APK sobre la instalación existente para sobrescribir los archivos del juego mientras conservas tu progreso.
</Callout>

</Section>
