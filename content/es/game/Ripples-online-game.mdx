<Section icon={Shield} title="Descarga Oficial y Pasos de Instalación">

Acceder al juego de forma segura es vital para proteger tu dispositivo de archivos comprometidos. El desarrollador distribuye todas las versiones oficiales exclusivamente a través de plataformas verificadas. Evita sitios web espejo de terceros o archivos APK no verificados, ya que a menudo contienen versiones desactualizadas o riesgos de seguridad.

El juego es compatible con el juego multiplataforma, lo que te permite transferir archivos de guardado entre sistemas de escritorio y móviles. A continuación, se presenta la guía verificada para descargar e instalar el juego de manera segura.

<Steps>
  <Step num="1" title="Visita la Página de Distribución Oficial">

  </Step>
  <Step num="2" title="Selecciona Tu Plataforma Compatible">
    Elige la versión adecuada para tu sistema operativo. El juego está optimizado para dispositivos Windows, macOS, Linux y Android.
  </Step>
  <Step num="3" title="Extrae e Instala los Archivos del Juego">
    Para las versiones de escritorio, extrae el archivo ZIP comprimido en una carpeta dedicada. Para usuarios de Android, descarga el archivo APK y asegúrate de que la configuración de tu dispositivo permita la instalación desde fuentes de navegador autorizadas.
  </Step>
  <Step num="4" title="Inicia y Configura los Ajustes">
    Abre el ejecutable o la aplicación. Navega al menú de preferencias para ajustar la velocidad del texto, la resolución y los niveles de audio para optimizar tu experiencia de lectura.

| macOS | DMG / ZIP | Arrastrar a Aplicaciones o ejecutar directamente |
| Linux | TAR.GZ | Extraer y ejecutar el script de inicio |
