<Section icon={Shield} title="Gerenciamento de Arquivos de Salvamento e Configuração de Plataforma">

Como esta visual novel é construída na engine Ren'Py, seus arquivos de salvamento são armazenados em pastas de diretório separadas dos arquivos principais do jogo. Esse design garante que, ao baixar e instalar uma nova versão, seu progresso seja detectado automaticamente. No entanto, migrações de plataforma ou instalações limpas às vezes podem correr o risco de perda de dados se não forem tratadas com cuidado.

Para garantir uma transição perfeita ao atualizar para a próxima versão, os jogadores devem se familiarizar com o processo de backup manual. Isso é particularmente importante para quem joga em vários dispositivos ou utiliza dispositivos Android.

<Steps>
  <Step num="1" title="Localize seu Diretório de Salvamento">
    No Windows, navegue até a pasta AppData digitando `%appdata%/RenPy` no explorador de arquivos. Localize a pasta associada ao jogo para encontrar seus dados persistentes.
  </Step>
  <Step num="2" title="Crie uma Cópia de Backup">
    Copie toda a pasta de salvamento para um local seguro em sua área de trabalho ou armazenamento em nuvem. Isso funciona como uma rede de segurança em caso de corrupção de arquivos durante o processo de atualização.
  </Step>
  <Step num="3" title="Baixe a Atualização">
    Assim que a próxima atualização for lançada oficialmente, baixe o pacote apropriado para o seu sistema operacional (Windows, macOS, Linux ou Android) na página oficial do itch.io.
  </Step>
  <Step num="4" title="Instale e Verifique">
    Extraia os novos arquivos do jogo e inicie o executável. O jogo lerá automaticamente seus arquivos de salvamento existentes, permitindo que você retome sua história imediatamente.
  </Step>
</Steps>

Para jogadores de dispositivos móveis, o gerenciamento de arquivos de salvamento requer precaução extra. Os sistemas operacionais Android frequentemente restringem o acesso aos diretórios internos de aplicativos, tornando os backups manuais um pouco mais complexos.

| Plataforma | Caminho de Localização do Salvamento | Ferramenta de Backup Recomendada |
| | | |
| Windows | `%appdata%/RenPy/Ripples` | Windows File Explorer |
| macOS | `~/Library/RenPy/Ripples` | Finder |
| Linux | `~/.renpy/Ripples` | Terminal / Gerenciador de Arquivos |
| Android | `Armazenamento Interno/Android/data` | Aplicativo Gerenciador de Arquivos Especializado |

<Callout type="warning" title="Aviso de Atualização no Android">
Ao atualizar o jogo no Android, não desinstale a versão anterior, a menos que seja explicitamente instruído pelo desenvolvedor. A desinstalação do aplicativo excluirá a pasta de dados correspondente, que contém seus arquivos de salvamento. Em vez disso, instale o novo APK sobre a instalação existente para substituir os arquivos do jogo enquanto preserva seu progresso.
</Callout>

</Section>
