<Section icon={Shield} title="Download Oficial e Passos de Instalação para Android">

Colocar o jogo para rodar na sua plataforma de preferência é simples. O desenvolvedor, Jestur, fornece versões otimizadas para sistemas operacionais de desktop, bem como para dispositivos móveis com Android. Como o jogo conta com mais de 10.000 imagens de alta qualidade, garantir que você tenha espaço de armazenamento adequado é crucial antes de iniciar o processo de instalação.

Siga estas instruções passo a passo para instalar o jogo na sua plataforma preferida.

<Steps>
  <Step num="1" title="Visite o Hub Oficial">
    Navegue até a página oficial do **Ripples itch io** ou o perfil do Patreon de Jestur. Isso garante que você está baixando a versão autêntica e não modificada do jogo.
  </Step>
  <Step num="2" title="Selecione a Versão da Sua Plataforma">
    Escolha o arquivo de download apropriado para o seu dispositivo. Usuários de desktop podem selecionar arquivos zip para Windows, macOS ou Linux, enquanto usuários de dispositivos móveis devem baixar o arquivo APK.
  </Step>
  <Step num="3" title="Extraia e Execute (PC/Mac)">
    Para sistemas desktop, extraia a pasta ZIP baixada em um diretório dedicado. Execute o arquivo executável (`Ripples.exe` no Windows ou o pacote de aplicativo no macOS) para iniciar o jogo.
  </Step>
  <Step num="4" title="Instale o APK (Android)">
    No seu dispositivo Android, ative a opção "Instalar de Fontes Desconhecidas" nas suas configurações de segurança. Localize o arquivo APK baixado usando um gerenciador de arquivos e toque nele para iniciar a instalação.
  </Step>
</Steps>
