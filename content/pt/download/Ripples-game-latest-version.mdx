<Section icon={Map} title="Download Oficial e Passos de Instalação">

Para desfrutar da melhor experiência de jogo, os jogadores devem baixar os arquivos oficiais diretamente de criadores verificados. Evite sites espelho de terceiros, que frequentemente distribuem versões desatualizadas ou arquivos comprometidos. O criador, Jestur, hospeda todos os lançamentos oficiais no itch.io e Patreon.

Instalar a **última versão do jogo Ripples** é simples, mas varia um pouco dependendo da plataforma escolhida. Siga as instruções passo a passo abaixo para configurar o jogo em seu sistema.

<Steps>
  <Step num="1" title="Visite as Plataformas Oficiais">
    Navegue até a página oficial do itch.io de Ripples ou visite o perfil de Jestur no Patreon. Estas são as únicas plataformas autorizadas onde o criador publica os pacotes oficiais do jogo.
  </Step>
  <Step num="2" title="Selecione a Versão para sua Plataforma">
    Escolha o pacote de download apropriado correspondente ao seu sistema operacional. O jogo oferece versões dedicadas para Windows, macOS, Linux e Android (formato APK).
  </Step>
  <Step num="3" title="Extraia os Arquivos do Jogo (PC/Mac)">
    Para as versões de desktop, baixe o arquivo ZIP compactado. Extraia o conteúdo para uma pasta dedicada em seu disco local usando um software padrão como WinRAR ou 7-Zip.
  </Step>
  <Step num="4" title="Inicie o Executável">
    Abra a pasta extraída. Clique duas vezes em `Ripples.exe` no Windows, ou execute o arquivo de aplicativo correspondente no macOS e Linux, para iniciar o jogo.
  </Step>
</Steps>

<Callout type="warning" title="Evite Versões Piratas">
Não baixe APKs modificados ou arquivos pré-extraídos de fóruns não oficiais. Esses arquivos podem corromper seus dados de salvamento ou expor seu dispositivo a vulnerabilidades de segurança. Sempre apoie o desenvolvedor nas plataformas oficiais.
</Callout>
