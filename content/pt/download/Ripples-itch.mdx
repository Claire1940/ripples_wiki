<Section icon={Target} title="Instalação Passo a Passo e Instalação Manual no Android">

A instalação de Ripples varia dependendo da plataforma escolhida. Enquanto os sistemas operacionais de desktop geralmente exigem apenas uma extração simples, os dispositivos Android envolvem a instalação manual de um pacote APK (sideloading).

Para garantir um processo de configuração tranquilo, siga estas etapas estruturadas de instalação para plataformas de desktop e móveis.

<Callout type="warning" title="Evite Fontes Não Oficiais">
Não baixe Ripples de espelhos de APK de terceiros, fóruns de compartilhamento de arquivos ou sites não autorizados. Esses arquivos frequentemente estão desatualizados, não têm suporte e podem conter softwares maliciosos que comprometem a segurança do seu dispositivo. Sempre use a página oficial do **Ripples no itch** ou o Patreon do Jestur.
</Callout>

<Steps>
  <Step num="1" title="Baixe o Pacote Correto">
    Navegue até a página oficial do itch.io ou Patreon. Escolha o pacote de download correspondente ao seu sistema operacional. Usuários de desktop devem baixar o arquivo compactado (ZIP ou TAR), enquanto os jogadores de dispositivos móveis devem baixar o arquivo APK.
  </Step>
  <Step num="2" title="Prepare a Instalação no Desktop">
    Para Windows, macOS ou Linux, localize o arquivo ZIP baixado. Extraia a pasta para um diretório dedicado em seu sistema (por exemplo, `C:\Jogos\Ripples`). Evite extrair diretamente em pastas temporárias, pois isso pode impedir a gravação correta dos arquivos de salvamento.
  </Step>
  <Step num="3" title="Ative a Instalação de Fontes Desconhecidas no Android">
    No seu dispositivo Android, abra as Configurações do sistema e navegue até Segurança ou Aplicativos. Ative a opção "Instalar aplicativos desconhecidos" ou "Permitir desta fonte" para o seu navegador de internet ou aplicativo gerenciador de arquivos.
  </Step>
  <Step num="4" title="Execute o Instalador e Inicie">
    No desktop, abra a pasta extraída e clique duas vezes no arquivo executável (como `Ripples.exe` no Windows ou `Ripples.app` no macOS). No Android, abra o seu gerenciador de arquivos, toque no arquivo APK baixado e siga as instruções na tela para concluir a instalação.
  </Step>
</Steps>

Gerenciar seus arquivos de jogo corretamente é essencial para manter seu progresso entre diferentes versões. Tabela abaixo descreve onde a engine Ren'Py armazena seus dados de salvamento em cada plataforma:

| Sistema Operacional | Diretório Padrão de Arquivos de Salvamento |

| macOS | `~/Library/RenPy/Ripples/` |
| Linux | `~/.renpy/Ripples/` |
| Android | `/Android/data/com.jestur.ripples/files/saves/` |
