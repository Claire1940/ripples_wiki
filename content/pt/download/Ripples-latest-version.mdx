<Section icon={Shield} title="Passos Oficiais de Download e Instalação">

Para manter a segurança do seu dispositivo e garantir a integridade dos seus arquivos de salvamento, obtenha sempre a última versão de Ripples por meio dos canais de distribuição oficiais. O criador hospeda os arquivos no itch.io para lançamentos públicos, enquanto os apoiadores do Patreon recebem builds de acesso antecipado. Evite baixar arquivos de mirrors de terceiros, sites de hospedagem de APK não verificados ou fóruns externos de compartilhamento de arquivos, pois esses arquivos podem ser modificados ou conter softwares maliciosos.

| Plataforma | Formato do Arquivo | Tamanho Médio do Arquivo | Método de Instalação |
| :--- | :--- | :--- | :--- |
| **Windows** | Arquivo ZIP | 1,8 GB | Extrair e executar o executável |
| **macOS** | DMG / ZIP | 1,9 GB | Arrastar para Aplicativos, ignorar o Gatekeeper |
| **Linux** | TAR.GZ | 1,8 GB | Extrair e executar o script shell |
| **Android** | APK | 1,2 GB | Instalação direta do pacote |

<Steps>
  <Step num="1" title="Visite o Portal Oficial">
    Navegue até a página oficial do itch.io de Ripples por Jestur ou acesse os links de download fornecidos através do seu portal de membros do Patreon.
  </Step>
  <Step num="2" title="Selecione a Build para sua Plataforma">
    Escolha o pacote apropriado para o seu sistema operacional. Certifique-se de ter pelo menos 3 GB de espaço de armazenamento livre para acomodar tanto o arquivo baixado quanto os recursos extraídos do jogo.
  </Step>
  <Step num="3" title="Extraia e Inicialize">
    Para versões de desktop, extraia o diretório compactado em uma pasta dedicada. Não execute o jogo diretamente de dentro do arquivo ZIP, pois isso impedirá que os arquivos de salvamento sejam gravados corretamente.
  </Step>
  <Step num="4" title="Configure as Permissões">
    No macOS e no Android, pode ser necessário conceder permissões de execução. Para o Android, ative a instalação de fontes desconhecidas nas suas configurações de segurança antes de iniciar o instalador.
  </Step>
</Steps>

<Callout type="warning" title="Segurança dos Arquivos de Salvamento">
Antes de instalar uma nova versão sobre uma build mais antiga, faça um backup manual do seu diretório de salvamentos. No Windows, os saves geralmente são armazenados no diretório AppData/Roaming, dentro da pasta local da RenPy.
</Callout>

</Section>

Ao atualizar para o pacote Android mais recente, os jogadores costumam encontrar problemas com a migração de saves. A tabela abaixo descreve os problemas técnicos mais comuns e as respectivas etapas para resolvê-los.

| Problema | Causa Raiz | Solução Recomendada |
| :--- | :--- | :--- |
| **Aplicativo Não Instalado** | Conflito de assinatura com a versão anterior | Desinstale a build mais antiga e, em seguida, instale o novo APK |
| **Tela Preta ao Iniciar** | Memória insuficiente ou recursos corrompidos | Limpe o cache do aplicativo e reinicie o dispositivo |
| **Arquivos de Salvamento Ausentes** | Alterações no diretório de armazenamento seguro (scoped storage) | Copie manualmente os arquivos de salvamento para a nova estrutura de diretório |
| **Dessincronização de Áudio** | Conflito de aceleração de hardware | Alterne as configurações de desempenho no menu do jogo |

<Callout type="success" title="Migração Manual de Saves">
Para transferir manualmente os saves no Android, use um gerenciador de arquivos para localizar o diretório de salvamento. Copie o conteúdo de `/Android/data/com.jestur.ripples/files/saves` para uma pasta de backup segura antes de realizar uma instalação limpa de uma nova atualização.
</Callout>
