<Section icon={Shield} title="Download Oficial e Etapas de Instalação">

Acessar o jogo com segurança é vital para proteger seu dispositivo contra arquivos comprometidos. O desenvolvedor distribui todas as versões oficiais exclusivamente por meio de plataformas verificadas. Evite sites espelho de terceiros ou arquivos APK não verificados, pois estes frequentemente contêm versões desatualizadas ou riscos de segurança.

O jogo suporta jogabilidade multiplataforma, permitindo que você transfira arquivos de salvamento (saves) entre sistemas de desktop e dispositivos móveis. Abaixo está o guia verificado para baixar e instalar o jogo com segurança.

<Steps>
  <Step num="1" title="Visite a Página de Distribuição Oficial">
    Navegue até a página oficial no itch.io ou o perfil do desenvolvedor no Patreon. Estas são as únicas plataformas autorizadas onde Jestur envia as versões genuínas do jogo.
  </Step>
  <Step num="2" title="Selecione Sua Plataforma Compatível">
    Escolha a versão apropriada para o seu sistema operacional. O jogo é otimizado para dispositivos Windows, macOS, Linux e Android.
  </Step>
  <Step num="3" title="Extraia e Instale os Arquivos do Jogo">
    Para versões de desktop, extraia o arquivo ZIP compactado em uma pasta dedicada. Para usuários do Android, baixe o arquivo APK e certifique-se de que as configurações do seu dispositivo permitam a instalação a partir de fontes de navegadores autorizadas.
  </Step>
  <Step num="4" title="Inicie e Configure as Opções">
    Abra o executável ou aplicativo. Navegue até o menu de preferências para ajustar a velocidade do texto, resolução e níveis de áudio para otimizar sua experiência de leitura.
  </Step>
</Steps>

| Plataforma | Tipo de Arquivo | Método de Instalação |
| :--- | :--- | :--- |
| Windows | Arquivo ZIP | Extrair e executar o arquivo .exe |
| macOS | DMG / ZIP | Arrastar para Aplicativos ou executar diretamente |
| Linux | TAR.GZ | Extrair e executar o script de execução |
| Android | APK | Habilitar fontes desconhecidas, abrir o APK para instalar |

<Callout type="warning" title="Alerta de Segurança">
Nunca baixe APKs modificados ou arquivos compactados de fóruns não oficiais. Essas versões não autorizadas podem corromper seus arquivos de salvamento e comprometer seus dados pessoais.
