<Section icon={Target} title="Guia de Instalação Passo a Passo">

A instalação adequada do Ripples 0.9.22 garante que os recursos do jogo sejam carregados corretamente e que o progresso existente na história seja preservado. Como o jogo é distribuído como um pacote independente, em vez de plataformas automatizadas, os jogadores devem gerenciar os arquivos manualmente. Isso é particularmente importante para usuários de Android e jogadores de desktop que estão migrando salvamentos de episódios anteriores.

O jogo é construído no motor Ren'Py, que armazena as configurações do usuário e os arquivos de salvamento em caminhos de diretório específicos de cada plataforma. Compreender esses locais evita a perda acidental de dados durante as atualizações de versão.

<Steps>
  <Step num="1" title="Baixe o Pacote">
    Visite a página oficial do itch.io ou o Patreon de Jestur. Selecione o arquivo compactado adequado correspondente ao seu sistema operacional. Certifique-se de que o download seja concluído totalmente antes de tentar abrir o pacote.
  </Step>
  <Step num="2" title="Prepare o Diretório">
    Para sistemas de desktop, crie um diretório limpo longe das pastas do sistema para evitar problemas de permissão. Para dispositivos Android, certifique-se de ter pelo menos 6 GB de espaço de armazenamento interno livre antes de começar.
  </Step>
  <Step num="3" title="Extraia ou Instale">
    No Windows, extraia o arquivo .ZIP usando um software de extração padrão. No Android, localize o arquivo .APK baixado usando um gerenciador de arquivos e toque nele para iniciar o instalador do pacote.
  </Step>
  <Step num="4" title="Configure as Permissões de Segurança">
    No macOS, você pode precisar contornar o Gatekeeper clicando com o botão direito no aplicativo e selecionando Abrir. No Android, permita a instalação de Fontes Desconhecidas nas suas configurações de segurança, se solicitado.
  </Step>
  <Step num="5" title="Verifique a Migração do Estado de Salvamento">
    Inicie o jogo. O motor deve detectar automaticamente os arquivos de salvamento existentes de locais de diretório anteriores. Verifique seu progresso carregando um ponto de salvamento recente a partir do menu principal.
  </Step>
</Steps>

| Requisito do Sistema | Especificação Mínima | Especificação Recomendada |
|---|---|---|
| Processador | Dual-Core 2.0 GHz | Quad-Core 2.5 GHz ou superior |
| Memória | 4 GB de RAM | 8 GB de RAM |
| Gráficos | Compatível com DirectX 9 | DirectX 11 / OpenGL ES 3.0 |
| Armazenamento | 6 GB de espaço disponível | 10 GB de espaço disponível (SSD preferencial) |
| SO | Windows 7 / Android 5.0 | Windows 10/11 / Android 11.0+ |

<Callout type="tip" title="Faça Backup dos seus Salvamentos">
Antes de atualizar para a versão 0.9.22, faça o backup manual do seu diretório de salvamento. No Windows, os salvamentos geralmente são encontrados na pasta AppData/Roaming/RenPy, enquanto no Android eles residem no diretório de armazenamento interno Android/data.
</Callout>
