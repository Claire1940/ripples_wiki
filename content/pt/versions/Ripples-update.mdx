<Section icon={Target} title="Como Instalar a Atualização de Ripples em Todas as Plataformas">

A instalação da última atualização requer o download da versão correta para o seu sistema operacional específico. O desenvolvedor oferece suporte oficial para Windows, macOS, Linux e Android. Para proteger seu sistema e apoiar o criador, sempre baixe os arquivos do jogo de fontes oficiais como o itch.io ou Patreon. Evite baixar arquivos de mirrors de terceiros não oficiais, pois estes podem conter malware ou arquivos de jogo corrompidos que podem quebrar seus arquivos de salvamento.

A tabela abaixo fornece os detalhes essenciais para cada plataforma suportada, incluindo os requisitos do sistema operacional e os locais padrão dos arquivos de salvamento.

| Plataforma | SO Suportado | Método de Instalação | Local do Arquivo de Salvamento |
| Windows | Windows 10/11 | Extrair o ZIP e executar o EXE | AppData/Local/Ripples |
| macOS | macOS Big Sur ou posterior | Extrair o DMG e arrastar para Aplicativos | Library/Application Support/Ripples |
| Linux | Baseado em Ubuntu/Debian | Extrair o tar.gz e executar o executável | .config/Ripples |
| Android | Android 8.0 ou superior | Instalar o APK oficial | Armazenamento Interno/Android/data |

Para jogadores mobile, a instalação da versão para Android requer alguns passos específicos para garantir que o jogo rode perfeitamente e acesse seu progresso existente. Siga este guia de instalação passo a passo para configurar a atualização no seu dispositivo Android.

<Steps>
  <Step num="1" title="Baixar o APK Oficial">
    Visite a página oficial no itch.io ou o Patreon no navegador do seu celular e baixe o arquivo APK mais recente do Episódio 9 para Android.
  </Step>
  <Step num="2" title="Ativar Fontes Desconhecidas">
    Navegue até as configurações do seu dispositivo Android, selecione Segurança ou Privacidade, e ative a permissão para instalar aplicativos de fontes desconhecidas.
  </Step>
  <Step num="3" title="Instalar e Substituir">
    Localize o APK baixado usando seu gerenciador de arquivos. Toque no arquivo para iniciar a instalação. Se você tiver uma versão anterior instalada, escolha atualizar e substituir o aplicativo existente para preservar seus dados de salvamento.
  </Step>
  <Step num="4" title="Verificar Permissões">
    Inicie o jogo e conceda as permissões de armazenamento se solicitado. Isso garante que o jogo possa ler e gravar arquivos de salvamento no armazenamento local do seu dispositivo.
  </Step>
</Steps>

<Callout type="warning" title="Evite a Perda de Arquivos de Salvamento">
Nunca desinstale a versão antiga do jogo no Android antes de instalar a atualização. Fazer uma desinstalação limpa apagará permanentemente sua pasta de salvamento local. Sempre realize uma instalação de substituição para manter seu progresso intacto.
</Callout>

<Section icon={Shield} title="Otimização Técnica e Resolução de Problemas">

Executar uma visual novel com mais de 10.000 imagens de alta qualidade pode ocasionalmente levar a problemas de desempenho, particularmente em dispositivos móveis com armazenamento ou RAM limitados. Garantir que seu dispositivo esteja devidamente otimizado evitará travamentos e falhas visuais durante o jogo.

Se você encontrar dificuldades técnicas após instalar a atualização mais recente, consulte a tabela de resolução de problemas abaixo para soluções rápidas para problemas comuns.

| Descrição do Problema | Causa Potencial | Solução Recomendada |
| O jogo trava ao iniciar | RAM insuficiente ou arquivos corrompidos | Limpar o cache, reiniciar o dispositivo ou baixar o pacote novamente |
| Arquivos de salvamento não carregam | Permissões ausentes ou mudança de diretório | Conceder acesso ao armazenamento manualmente nas configurações do aplicativo |
| Imagens de fundo ausentes | Extração incompleta de arquivos ZIP | Extrair novamente os arquivos do jogo usando uma ferramenta oficial como o 7-Zip |

Para manter o desempenho ideal em todas as plataformas, considere implementar as seguintes melhores práticas:

- Feche aplicativos em segundo plano para liberar memória RAM antes de iniciar o jogo.
- Certifique-se de que seu dispositivo tenha pelo menos 3 GB de espaço de armazenamento interno livre para acomodar o cache de imagens de alta resolução.
- Faça backup regularmente do seu diretório de salvamento em um dispositivo de armazenamento externo ou serviço em nuvem.

Ao seguir estas diretrizes técnicas, você garantirá uma sessão de jogo estável e de alto desempenho, permitindo que você mergulhe totalmente na história de Jack e Darci sem interrupções técnicas.
</Callout>
