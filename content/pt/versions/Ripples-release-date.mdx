<Section icon={Target} title="Como Baixar e Atualizar">

A transição entre episódios requer práticas de instalação adequadas para evitar a perda das escolhas do jogador. Como *Ripples* depende de decisões tomadas nos capítulos iniciais para determinar os resultados no final do jogo, preservar seus arquivos de save é fundamental.

<Steps>
  <Step num="1" title="Visite as Páginas de Distribuição Oficiais">
    Navegue até a página oficial do itch.io para Ripples ou acesse a página do Patreon do criador. Os apoiadores do Patreon frequentemente recebem acesso antecipado às versões futuras antes de serem lançadas para o público geral no itch.io.
  </Step>
  <Step num="2" title="Selecione a Versão Correta">
    Baixe o pacote específico projetado para o seu sistema operacional. Certifique-se de ter espaço de armazenamento suficiente, pois o tamanho dos arquivos aumenta a cada novo episódio.
  </Step>
  <Step num="3" title="Extraia e Instale os Arquivos do Jogo">
    Para plataformas de PC, extraia a pasta compactada baixada para um diretório dedicado. Não execute o jogo diretamente de dentro de uma pasta compactada, pois isso impede que os arquivos de save sejam gravados corretamente. Para Android, habilite a instalação de fontes desconhecidas nas configurações do seu dispositivo antes de abrir o APK.
  </Step>
  <Step num="4" title="Verifique a Migração dos Arquivos de Save">
    O motor do jogo detecta automaticamente as pastas de save existentes no diretório padrão do sistema. Inicie o novo episódio e verifique o menu de carregamento para confirmar que seu progresso dos episódios anteriores foi transferido com sucesso.
  </Step>
</Steps>

<Callout type="tip" title="Recomendação de Backup Manual">
Antes de instalar uma nova atualização de episódio, copie manualmente seu diretório de saves para uma pasta de backup segura. No Windows, os dados de save geralmente estão localizados no diretório AppData/Roaming/RenPy sob o nome exclusivo da pasta do jogo.
</Callout>

</Section>
